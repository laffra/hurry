Hurry adds a timer to your Google Slides presentation. Add something like this to your speaker notes in slide 1: 

        # time=45
        # color=pink
Here are the elements you can use:
Name Type Explanation
time number Total number of minutes the presentation lasts.
color string The color of the progress bar when there is plenty of time left.
warning string The color of the progress bar when there half of the time is gone.
errro string The color of the progress bar when you really need to hurry.
